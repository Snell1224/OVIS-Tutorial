Fiscal Year 2021 LDMS Basic Tutorial Videos

The following videos are live examples of each of the exercises described in the LDMS Basics Tutorial Powerpoint (under Resources).

The tutorials are performed in a container which has been setup with the directory structure described in the Powerpoint (slide 14). The LDMS version 4.?? is installed within this container so all video exercises listed below were performed with this version.

Container access is provided to all attendees for LDMSCON2022 Website.

Instructions on how to access the container will be given after the conference (June 13th-15th). If you still wish to go through the videos without container access, then please visit LDMS's Quick Start webpage for further instructions on how to build and install LDMS on your computer.

Please refer to this powerpoint or the man pages in the LDMSD github for help and information about the sampler plugins.
